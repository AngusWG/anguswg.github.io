# windows 下使用 make

- 下载 [cygwin](http://www.cygwin.com/)
- 安装时选择包 make
- 将`C:\cygwin64\bin`添加到系统环境变量第一条 ，放后面会有 bug.
- 然后就可以用了

- 有挺多 Linux 命令的，在没有将 wsl2 作为主力平台的时候，cygwin 的挺多命令能弥补 windows 的部分不足。
